<b>Icon Tester</b>
===========
Icon Test Tool for Designer<br>
Icon Tester is Web App, you will be able to test easily your icon by Add to Home Screen function on iOS - Safari or Android - Google Chrome.<br>
To use Icon Tester you will need your Web Server(Local or Internet).<br>
<i>Icon Tester Ver.2.1 — Support iOS, Android</i><br>
http://daramghaus.com/icontester<br>

<b>How To Use</b><br>
<ol>
	<li><b><a href="http://daramghaus.com/icontester/download" target="_blank">Download</a></b> Icon Tester to your Mac or PC.</li>
	<li>Unzip and Upload all files(index.html, css and images folders) to your Web Server.</li>
	<li>Overwrite your icon image files on <b>images</b> folder.<br><b>Android</b>: Overwrite "<i>images/android/icon.png</i>" image.</li>
	<li>Open the <b>index.html</b> on device default web browser of your device.<br>
	<b>iOS</b>: Safari(iPhone or iPad)<br>
	<b>Android</b>: Google Chrome</li>
	<li>Add to Home Screen.<br>
	<b>iOS</b>: Tap <span class="share_icon_inline"></span><b>Share</b> bottun and then <span class="share_add_icon_inline"></span><b>Add to Home Screen</b>
	<br><img src="http://static.tumblr.com/iawroni/PFrn5vg34/info.jpg" style="width:100%;height:auto;padding:8px 0 8px">
	<b>Android</b>: Tap menu button(top right), Tap <b>Add to Home Screen</b>.
	<br><img src="http://static.tumblr.com/iawroni/o2kn5vgrk/info_android.jpg" style="width:100%;height:auto;padding:8px 0 8px">
	</li>
	<li>If you want Icon refresh, Overwrite icon image files again,<br>
	<b>iOS</b>: Run and close the icon on Home Screen, Refresh automatically.<br>
	<b>Android</b>: Delete Icon on Home Screen, and add icon again on Chrome.</li>
</ol>
